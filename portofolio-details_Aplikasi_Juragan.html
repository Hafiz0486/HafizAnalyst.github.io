Aplikasi Buku Besar Berbasis Delphi 7

            

                Today's traders, if they want to increase their trading, users need documents that can record all the trading processes that occur. This documentation is called the Ledger which records everything in nominal form which is divided into debits and credits, but rarely do traders who start trading know about this. Therefore, the author tried a simple program to record trade and ledger information. Trade information consists of manager or producer, goods and consumer data and the general ledger consists of journals, adjusting journals, track balances, reports and report documentation. This program is based on Delphi 7 which uses Microsoft Access as its DBMS. This program helps traders record information simply and will help users to process it with several buttons provided.
